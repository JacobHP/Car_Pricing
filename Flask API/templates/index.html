Enter values as follows:
• If your car brand is not on the list, please select "Other"
• Engine is measured in litres, please enter to the nearest 1 decimal place
• Power is measured in BHP. Conversions can be found here
• If there are any issues with the car (structural damage, non-runner etc.) please select 'Any Issues?' as 'Y'

This was a starter regression project using xgboost. The model works well for normal-priced cars (less than £100k).
However as expected can be volatile for higher priced cars. Full details of the project can be found here.
